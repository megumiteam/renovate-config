# @digitalcube/renovate-config

## Usage

### Default

```json
{
    "extends": ["@digitalcube"]
}
```


### Demo applications

```json
{
    "extends": ["@digitalcube:examples"]
}
```

## Development


```bash
$ git clone git@github.com:megumiteam/renovate-config.git
$ cd renovate-config
$ npm install

$ npm run test
```
